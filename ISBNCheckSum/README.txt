How to use the program:
	1. Click the button that says "Get 10 Digit ISBN numbers"
		This will read the ISBN numbers from the file
		from ISBNCheckSum/bin/Debug/isbn9.txt
		Then it will create the output in ISBNCheckSum/bin/Debug/isbn10Output.txt
	2. Open the output file
		The numbers on the output file coorespond to the numbers on the input file.
		The final digit in each row of output is the checksum value.
		The first ISBN number is the ISBN for our textbook for the class.
